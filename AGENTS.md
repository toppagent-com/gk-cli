# AGENTS.md

This file provides guidance to AI agents working in this repository.

## Project Overview

This is the public repository for **GitKraken CLI** (`gk`), a command-line interface for GitKraken that enables multi-repo workflows, AI-powered commit messages, pull request generation, and an MCP server for git and issue/PR hosting providers. The repository contains documentation, issue templates, and community resources -- not the CLI source code itself.

## Repository Structure

- `README.md` — Primary documentation for the CLI: installation, usage, workflows, configuration, and troubleshooting.
- `CONTRIBUTING.md` — Contribution guidelines for bug reports and feature requests.
- `LICENSE` — Project license (Creative Commons Attribution 3.0 US for documentation).
- `.github/ISSUE_TEMPLATE/` — GitHub issue templates for bug reports and feature requests.
- `images/` — Image assets referenced by the README.

## Guidelines

### Editing Documentation

- The README is the main deliverable of this repo. Keep it well-structured, accurate, and consistent with the existing table of contents.
- When adding new sections to the README, also add a corresponding entry to the Table of Contents.
- Use fenced code blocks with `bash` language tags for CLI examples.
- Keep instructions platform-aware (macOS, Windows, Unix/Ubuntu) where relevant.

### Issues and Contributions

- Follow the existing issue templates in `.github/ISSUE_TEMPLATE/` when creating issues.
- Refer to `CONTRIBUTING.md` for contribution norms.

### Style

- Use clear, concise technical writing.
- Maintain consistent Markdown formatting (ATX-style headings, fenced code blocks).
- Do not add unnecessary files; this is a lean documentation repository.
