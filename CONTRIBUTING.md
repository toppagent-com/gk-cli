## Contributing

Hello! Thank you for your interest in the GitKraken CLI. We love feedback from our users!

We accept feedback for bug fixes and features. We'd also love to hear about ideas for new features.

Please do:

* Check existing issues to verify that the bug or feature request has not already been submitted.
* Open an issue if things aren't working as expected.
* Open an issue to propose a significant change.
* Use a clear and descriptive title for issues and pull requests.
* Provide as much relevant information as possible when reporting a bug, including steps to reproduce, expected behavior, and actual behavior.
* Fork the repository and create a feature branch for your changes.
* Keep pull requests focused on a single change to make them easier to review.
* Write clear, concise commit messages that explain the "why" behind a change.
